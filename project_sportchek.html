Sport Chek CMS - The situation

THE SITUATION

The task was to create a Custom CMS System for a fictitious Sport Chek website. This is a project developed in the back-end class and this representation is only for functionality purposes, the design part hasn’t been developed yet. Some of the instructions for this project were to create functions that would allow the user to manage, customize and add products to their already-existing website.

Sport Chek CMS - The solution

THE SOLUTION

The result is an app created mainly with PHP that holds a MySQL database in which the products with all the details (such as name, image, description, price and category) are saved. The regular user can filter the content based on the product’s category, and they can also click a button that takes them to see more details of each item.
