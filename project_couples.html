COUPLES RESORT

Web Designer

Mixtape Project - The situation

THE SITUATION

The challenge for this project was to take an already-existing website for the Couples Resort company, analyze the main features, values, ideas, colours, target, and so on to re-design the landing page for a more aesthetic and fresher look.

Mixtape Project - The solution

THE SOLUTION

The result is a stylish website that receives the user with a full-width image of the beautiful hotel installations. There is a button at the very top of the website that invites the user to book a room, which is one of the main reasons why the users visit their website. The second reason? To see images and activities, which brings us the next section of the website that describes some of the activities that they offer at the resort and so on. The imagery used for this website can be described as professional, fancy, romantic, warm and of high-definition. The website contains a carrousel with the different types of rooms that the resort has to offer, tree key points to compare with the rest and a button that takes the user to another page with all the details of the room, pictures and reservation forms. In the end, the site has a section where some real testimonials are displayed.
