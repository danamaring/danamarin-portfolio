ChatApp Project - The situation

THE SITUATION

The task was to design and develop an interactive app with socket.io and Vue.js that binds data to the views. The project should display two users (each of them using a different browser) that would communicate through the app. From the design standpoint, the user should be able to differentiate between the messages that they send and the ones that they receive. Another big component of this project was the functionality of the username that would also help distinguish between both sides of the chat.

ChatApp Project - The solution

THE SOLUTION

The result is a fully responsive and interactive app that asks the user to set a username on the landing page, it keeps the value as an array and then displays an engaging interface that allows them to maintain chats with whoever logs in. As mentioned before, this app works with the socket.io and Vue.js libraries and it binds data through the browser and the respective components.
